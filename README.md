To-Do List App (Python CLI)

 Description

A simple command-line To-Do List application that allows users to manage tasks efficiently. Tasks are stored in a text file for persistence.

 Features

* Add new tasks
* View all tasks
* Mark tasks as completed
* Delete tasks
* Persistent storage using a file

 Technologies Used

* Python
* File Handling

 How to Run

1. Make sure Python is installed.
2. Save the file as `todo.py`
3. Run the program:

   ```bash
   python todo.py
   ```

File Storage

Tasks are stored in `tasks.txt` automatically.
